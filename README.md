Clustering App

This project is a Flask web application for clustering image datasets.

It allows users to:
- Upload image datasets (ZIP files)
- Extract image features using a pretrained MobileNetV2 model
- Cluster images using K-Means
- Evaluate clustering quality
- Visualize clusters and distributions

The system is used for basic unsupervised learning experiments on image data.

How to run:
1. pip install -r requirements.txt
2. python app.py
3. open http://127.0.0.1:5000
